# JUnit Exercises

Covers the mandatory JUnit hands-on:
- **Exercise 1 – Setting Up JUnit:** add the JUnit dependency to `pom.xml` (see below).
- **Exercise 3 – Assertions:** `AssertionsTest.java`
- **Exercise 4 – AAA pattern + setup/teardown:** `CalculatorTest.java`

## Exercise 1: Maven dependency
Add this to your project's `pom.xml`:

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```
